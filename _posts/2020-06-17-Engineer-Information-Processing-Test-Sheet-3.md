## 상세 설명

① 메소드

JAVA에서 특정 기능을 수행하는 함수를 지칭하는 단어

동작이나 연산으로 생각하면 이해하기 쉽다.

② 클래스

공통된 속성과 연산(행위)을 갖는 객체의 집합

③ 상속성

이미 정의된 상위 클래스의 모든 속성과 연산을 하위 클래스가 물려받는 것

④ 매세지

객체들 간에 상호작용을 하는데 사용되는 수단으로 객체의 메소드(동작,연산)를 일으키는 외부의 요구 사항
